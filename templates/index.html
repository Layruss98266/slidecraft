SlideCraft Help Guide

Everything you need to know to get started

Getting Started

1
Upload a PPTX
Click Upload in the header or drag & drop a .pptx file anywhere on the page.
2
Remove NotebookLM Logo
Click Remove Logo to seamlessly erase the watermark from all slides using AI inpainting.
3
Edit & Export
Add overlays, edit text, apply filters, then click PPTX, PDF, PNG, or GIF to export.

Editing Text in Slides

A
OCR Method (Automatic)
Click OCR in the toolbar. Text regions appear as blue dashed boxes. Click any box to cover the original and type replacement text. Adjust font, size, color in the Properties panel. Click Apply to burn changes into the image.
B
Cover & Type (Manual)
Press C or select the Cover tool. Draw a rectangle over text you want to replace. The background color is auto-sampled. Type your new text, then click Apply.
C
Inline Editing
Double-click any text overlay to edit it directly on the canvas.

Drawing & Shapes

S Select — Click to select, drag to move, handles to resize
T Text — Draw a text box, type content
R Rectangle — Draw a colored rectangle
C Cover — Cover original text and replace
Circle — Draw circles and ellipses
Line — Draw lines with arrowheads
Callout — Speech bubble callout shapes
Freehand — Draw freely with pen tool
Image — Upload and place images as overlays

Toolbar Guide

TOOLS — Select, Text, Rect, Circle, Line, Callout, Draw, Cover
EDIT — Undo, Redo, Delete, Duplicate
INSERT — Image, QR Code, Watermark
SLIDE — OCR, Crop, Rotate, Snap Grid
MANAGE — Find/Replace, History, Templates, Comments

Right Panel Tabs

Layers — View and select all overlays on the current slide
Properties — Edit font, color, size, effects of selected overlay
Notes — Add speaker notes (included in PPTX export)
Filters — Apply brightness, contrast, blur, grayscale, sepia to slides
Comments — View, resolve, and delete annotation comments

Export Options

PPTX — PowerPoint with overlays as editable text layers
PDF — Multi-page PDF of all slides
PNG — ZIP archive of all slides as PNG images
GIF — Animated slideshow GIF
Apply — Burns overlays permanently into the slide images

Video Logo Remover

1
Click Video in the header to open the video tool
2
Upload your video (MP4, MOV, AVI, MKV)
3
Drag the red box over the logo on the preview frame
4
Click Remove Logo — watch real-time progress with ETA
5
Click Stop to cancel, or Download when done

Tips & Tricks

Press ? to see all keyboard shortcuts
Press F5 for fullscreen presentation mode
Press G to toggle the alignment snap grid
Ctrl+Z / Ctrl+Y for undo/redo (50 steps)
Ctrl+C / Ctrl+V to copy/paste overlays across slides
Right-click an overlay for context menu (duplicate, front/back)
Drag thumbnails in the sidebar to reorder slides
Save versions regularly — use History to rollback if needed
Save Templates to reuse slide layouts across presentations

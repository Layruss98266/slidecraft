SlideCraft guide

SlideCraft is a free, local slide editor that runs in your browser. It was built specifically for editing image-based presentations like those exported from NotebookLM, Gamma, or similar tools. You can fix text, remove watermarks, add shapes, apply filters, and export a clean PPTX file. Everything runs on your own machine, so nothing leaves your computer.

Python 3.10+ Flask 3.x Runs on your machine No build step needed No cloud, no subscription

Before you install

You only need three things before getting started. Here is what each one does and how to check if you already have it.

  • Python 3.10 or newer. This is the programming language SlideCraft is built on. Open a terminal and type python --version to check. If it says 3.10 or higher, you are good. If not, download it from python.org. On Windows, make sure to tick the "Add Python to PATH" checkbox during installation or things will not work properly.
    On macOS and Linux you may need to type python3 --version instead.
  • pip (Python's package installer). This is used to install SlideCraft's dependencies. It comes bundled with Python automatically. Check it works by running pip --version.
  • LibreOffice (strongly recommended). SlideCraft uses LibreOffice to convert your PPTX slides into images. Without it, slides will look incomplete since only embedded pictures will be extracted and all text and shapes will be missing. Download it for free from libreoffice.org.
LibreOffice is technically optional but you will notice a significant difference in slide quality with it installed. It takes about 5 minutes to set up and is worth it.

Installation

The setup takes about 5 to 10 minutes the first time, mostly waiting for packages to download. Follow these steps in order.

Step 1: Get the code

If you have Git installed, run this in your terminal:

git clone https://github.com/Layruss98266/slidecraft.git
cd slidecraft

If you do not have Git, that is fine. Go to the GitHub page, click the green Code button, choose Download ZIP, then extract the folder somewhere on your computer and open a terminal inside it.

Step 2: Create a virtual environment

A virtual environment keeps SlideCraft's packages separate from everything else on your computer. It is good practice and prevents version conflicts. Run the command for your operating system:

python -m venv .venv
.venv\Scripts\activate
python3 -m venv .venv
source .venv/bin/activate
python3 -m venv .venv
source .venv/bin/activate

Once activated, you will see (.venv) at the beginning of your terminal prompt. You need to activate it every time you open a new terminal before running the app.

Step 3: Install the dependencies

pip install -r requirements.txt

This downloads everything SlideCraft needs to run. It only needs to happen once. The first install takes a few minutes because some packages like PyTorch (used for OCR) are quite large.

If pip gives you an error, try python -m pip install -r requirements.txt instead. Some systems require the longer form.

Starting the app

The easy way (recommended)

Use the launcher script included in the project. It handles activating the virtual environment and starting the server for you automatically:

run.bat
.\run.ps1
./run.sh

The manual way

If you prefer to start it yourself, activate your virtual environment first, then run:

python app.py

Once you see the message Running on http://127.0.0.1:5050, open your browser and go to that address. Press Ctrl+C in the terminal whenever you want to stop the server.

Your slides, overlays, and version history are saved to disk. Stopping and restarting the server will not lose your work.

Uploading slides

To load a presentation, either click the Upload button in the header or simply drag a .pptx file anywhere onto the editor window. SlideCraft will convert each slide into an image and display them in the strip on the left.

The conversion quality depends on what software you have installed. SlideCraft tries each method in this order:

  • LibreOffice + pdf2image (best quality). Converts the PPTX to a PDF first using LibreOffice, then renders each page as a high-resolution image. Fonts, shadows, and gradients all look exactly as they do in PowerPoint.
  • LibreOffice + PyMuPDF (very good quality). Same first step but uses a different library to render the PDF. Used as a fallback if pdf2image is not available.
  • Pillow extraction (limited). Used only when LibreOffice is not installed. It pulls out the embedded images from the PPTX file but cannot capture text or drawn shapes. Your slides may look incomplete.
You can also upload up to 20 PPTX files at once using the batch upload feature. SlideCraft will remove logos from all of them and give you a ZIP file to download.

Drawing tools

You can place various shapes and elements on top of any slide. Everything you add is called an "overlay." Overlays sit on top of the slide image and can be moved, resized, or deleted at any time before you bake them in.

Text box

Add styled text anywhere. Control font, size, color, weight, shadow, outline, and spacing. Press T to activate.

Rectangle

Draw a filled rectangle. Good for highlighting areas or covering content. Press R.

Circle

Draw circles and ellipses with a fill color of your choice.

Arrow and line

Draw straight lines, with or without arrowheads. Great for pointing to things.

Callout bubble

A speech bubble shape for adding comments or annotations to specific areas.

Freehand

Draw anything with your mouse. Useful for circling things or adding a signature.

Cover and type

Draw a box over text you want to replace, then type the new text on top. Press C.

Image overlay

Upload a photo, logo, or icon and place it anywhere on the slide. Adjust size and opacity.

Every overlay can be resized using the 8 handles around its edges, moved by dragging, and right-clicked for options like duplicate, delete, or changing its layer order. You get 50 steps of undo history per slide with Ctrl+Z.

Editing text with OCR

OCR stands for Optical Character Recognition. SlideCraft uses it to detect the text that is already baked into your slide images, so you can click on it and replace it without needing the original source file.

  • Click the OCR button in the toolbar. SlideCraft scans the slide and draws blue dashed boxes around every piece of text it detects. This may take a few seconds.
    The first time you click OCR, it downloads the detection model (~100 MB). After that, it is instant.
  • Click on any blue box to select that text region. SlideCraft automatically samples the background color, text color, and font weight from that area and creates a cover overlay that matches the original style.
  • Edit the text in the Properties panel on the right. Change the content, adjust the font, size, color, alignment, or letter spacing until it looks right.
  • Click Apply when you are done. This permanently bakes all your overlays into the slide image. After this point you cannot undo individual changes to that slide, so save a version first if you want a safety net.
Clicking Apply is permanent for that slide. Before you apply, use Save Version from the History panel if you want to be able to go back.

Removing logos and watermarks

Remove an existing watermark

SlideCraft uses OpenCV's inpainting algorithm to fill in the area where the logo was, sampling from the surrounding pixels to make it look seamless. It works well on simple backgrounds like gradients and solid colors.

  • Click Remove Logo in the header to strip the NotebookLM watermark from every slide in one go. This is the quickest option when you know exactly where the logo is.
  • Or use Detect Watermark for more control. SlideCraft scans the slide corners using edge analysis to locate the logo automatically. You can then choose to remove it from just the current slide or all slides at once.

Add your own watermark

You can also stamp your own branding onto all slides at once using the Watermark tool. Add a text label like "CONFIDENTIAL" or "DRAFT", or upload a logo image. You can control where it appears (center, any corner, or tiled across the slide), how transparent it is (5% to 80%), and how large it is. A live preview shows you what it will look like before you commit.

Removing backgrounds from images

If you have uploaded a logo or photo as an image overlay and it has a white or solid-colored background, you can remove it to get a transparent cutout. This is useful when you want your logo to sit cleanly on top of a slide without an ugly white box behind it.

  • Add an image overlay to your slide using the image tool in the toolbar, or drag an image file directly onto the slide.
  • Click the image to select it. You will see a blue border appear around it.
  • Click Remove BG in the toolbar. The background is removed using an AI model called U2Net (built into the rembg package). The result is a transparent PNG with just your subject remaining.
    First use downloads the U2Net model (~170 MB). You can undo the result with Ctrl+Z if you are not happy with it.

Image filters

Filters let you adjust the look of the entire slide image. Open the Filters tab in the right panel to access them. They apply instantly so you can see the result straight away, and they are non-destructive until you click Apply.

Brightness

Make the whole slide lighter or darker.

Contrast

Increase the difference between light and dark areas, or flatten it out.

Saturation

Boost the color intensity or tone it down toward gray.

Blur

Apply a soft Gaussian blur to the slide.

Sharpen

Add crispness to a soft or compressed slide image.

Grayscale

Convert the slide to black and white.

Sepia

Give the slide a warm, vintage brown tone.

Reset

Undo all filter changes and return to the original image.

Templates and version history

Version history

Think of this as a manual save point system. Before you do something that cannot be undone, like applying overlays, removing a logo, or cropping a slide, click Save Version in the History panel. SlideCraft takes a snapshot of everything in its current state. If something goes wrong, you can go back to that snapshot by clicking Restore.

Templates

If you work with a consistent slide style and want to reuse it, save your current deck as a named template. This saves the slides and all overlays together. You can load that template into any new session to start from a ready-made layout instead of from scratch.

Master slide

The master slide lets you apply consistent branding across your entire deck in one action. Set a header, footer, page numbers, and brand colors and they will propagate to every slide automatically.

Auto-save

SlideCraft saves your overlay changes automatically every 8 seconds in the background. You will see a pulsing orange dot in the header when there are unsaved changes, and a green checkmark when everything is up to date. You do not need to manually save as you work.

AI tools

SlideCraft has four AI-powered tools for editing and generating slide content. They all run locally through Ollama, which means no API keys, no data sent to the cloud, and no usage fees.

Setting up Ollama

  • Download and install Ollama from ollama.com. It works on Windows, Mac, and Linux.
  • Download a language model. Open a terminal and run: ollama pull llama3. This downloads the model (a few gigabytes) and only needs to happen once.
  • Make sure Ollama is running before using any AI button in SlideCraft. You can start it by running ollama serve in a terminal, or it may already be running in your system tray after installation.

AI Rewrite

Select a text overlay and click AI Rewrite. It will rewrite the content to be more concise or professional while keeping the meaning.

AI Translate

Translate the selected text overlay into any language. Useful for localizing slides.

AI Alt-text

Generates a plain-English description of the current slide for accessibility purposes.

AI Slides

Type a prompt and SlideCraft will generate slide content and a layout for you.

Presenting your slides

SlideCraft has a built-in fullscreen presentation mode so you can show your slides directly from the editor without exporting first.

  • Press F5 to enter fullscreen presentation mode.
  • Use Left Arrow and Right Arrow to move between slides.
  • Press L to enable a red laser pointer dot that follows your cursor.
  • Slides transition with fade, slide, or zoom effects depending on your settings.
  • If you have uploaded audio narration for a slide (MP3 or WAV), it will play automatically.
  • YouTube, Vimeo, and Loom video embeds will play inline during the presentation.
  • Press Esc to go back to the editor.

Exporting your work

When you are happy with your slides, use the Export button in the toolbar to download them. Choose the format that works best for your situation:

PPTX
PowerPoint file
Exports a .pptx file with your slides as images and all text overlays as real editable text layers. Opens in PowerPoint, LibreOffice, or Google Slides. This is the best choice if you need to keep editing the file elsewhere.
PDF
PDF document
Creates a multi-page PDF with one slide per page. Great for sharing with people who just need to view the content, or for printing.
PNG
Image files (ZIP)
Downloads a ZIP file containing one full-resolution PNG image for each slide. Useful if you need to upload individual slide images somewhere else.
GIF
Animated GIF
Creates a looping animated GIF of all slides. You can set how long each slide stays on screen. Good for social media or embedding in docs.
Apply
Bake overlays into slides
This is not an export format. Clicking Apply permanently merges all your overlays into the slide images so there are no separate layers. Do this before PNG or GIF export if you want everything merged into a single flat image per slide.
.slidecraft
Portable archive
A single file that contains everything: slides, overlays, comments, and master slide settings. You can import it into any SlideCraft instance to pick up exactly where you left off.

Configuration

These settings are all optional. The defaults work fine for most people. If you need to change any of them, set the environment variable before running python app.py.

SettingDefaultWhat it does
PORT5050The port number the server listens on. Change this if port 5050 is already in use on your machine.
HOST127.0.0.1Set this to 0.0.0.0 if you want other devices on your local network to be able to open the editor. Only do this on a trusted network since there is no login by default.
FLASK_DEBUGfalseSet to true during development. The server will automatically restart whenever you change the code.
MAX_UPLOAD_MB1024The maximum total upload size in megabytes. The default is 1 GB which is plenty for most uses. Raise it if you are batch uploading very large files with embedded video.
EXPORT_TTL_SECONDS86400How many seconds exported files are kept before being deleted automatically. The default is 24 hours.

How to set them on Windows (PowerShell)

$env:PORT = "8080"
$env:HOST = "0.0.0.0"
python app.py

How to set them on macOS and Linux

PORT=8080 HOST=0.0.0.0 python app.py

Troubleshooting

If something is not working as expected, check the list below. Most issues have a quick fix.

Slides look blank or only show some images after upload
This happens when LibreOffice is not installed. Without it, SlideCraft can only extract the images that are embedded directly in the PPTX file. Any text and shapes drawn in PowerPoint will be invisible. Install LibreOffice from libreoffice.org and try uploading again.
Clicked OCR but nothing happened
The first time you use OCR, SlideCraft needs to download the text detection model which is about 100 MB. This happens in the background and can take a minute or two. Make sure your internet connection is active, wait a moment, and try again. A loading spinner should appear while the download is in progress.
Getting a UnicodeEncodeError on Windows
This is a Windows terminal encoding issue. Fix it by setting the encoding before starting the server. In Command Prompt: set PYTHONIOENCODING=utf-8 && python app.py. In PowerShell: $env:PYTHONIOENCODING="utf-8"; python app.py.
Remove BG says "rembg not installed"
Make sure your virtual environment is activated, then run pip install rembg and restart the server. If rembg is already in requirements.txt and the install seemed to work, check that you are running the app from within the same virtual environment where you installed it.
Video logo removal is taking a very long time
This is normal. SlideCraft processes every single frame individually using the CPU, which is slow for long or high-resolution videos. A 5-minute 1080p video typically takes between 5 and 8 minutes to process. The progress bar will show you how far along it is.
QR code shows a grey box instead of a real code
The QR code feature requires an extra package that is not installed by default. Run pip install qrcode[pil] in your activated virtual environment, then restart the server.
Getting a 413 error when uploading
The file or batch of files you are trying to upload exceeds the current size limit. Start the server with a higher limit: MAX_UPLOAD_MB=2048 python app.py. The default is already 1 GB so this mostly comes up with very large video-heavy batches.
The replacement text font does not match the original
SlideCraft tries to match the font automatically using OCR, but it can only use fonts that are installed on your operating system. Open the Font Family dropdown in the Properties panel and try different options until you find the closest match to the original.
AI buttons are not responding or returning errors
The AI tools need Ollama to be running in the background. Check that it is installed, that you have pulled at least one model with ollama pull llama3, and that the Ollama service is active. You can start it manually by running ollama serve in a terminal.

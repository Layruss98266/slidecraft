## What is SlideCraft?

SlideCraft is a web-based PowerPoint slide editor built for editing **image-based presentations** (like those generated by NotebookLM, Gamma, or similar AI tools). It lets you:

- **Edit text directly on slide images** using OCR detection
- **Remove NotebookLM watermarks** from slides and videos with one click
- **Add text, shapes, and image overlays** with a full-featured editor
- **Export as PPTX or PDF** with all edits baked in

Built with Flask + vanilla JS. No React, no build step. Just `python app.py` and go.

## Installation (Step by Step)

### Step 1: Install Python

SlideCraft requires **Python 3.10 or higher**.

<details>
<summary><strong>Windows</strong></summary>

1. Go to [python.org/downloads](https://www.python.org/downloads/)
2. Download the latest Python installer (e.g. Python 3.12.x)
3. Run the installer
4. **IMPORTANT**: Check the box **"Add Python to PATH"** at the bottom of the installer
5. Click **"Install Now"**
6. Verify installation — open Command Prompt and run:
   ```
   python --version
   ```
   You should see something like `Python 3.12.x`

</details>

<details>
<summary><strong>macOS</strong></summary>

1. Install [Homebrew](https://brew.sh/) if you don't have it:
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```
2. Install Python:
   ```bash
   brew install python
   ```
3. Verify:
   ```bash
   python3 --version
   ```

</details>

<details>
<summary><strong>Linux (Ubuntu/Debian)</strong></summary>

```bash
sudo apt update
sudo apt install python3 python3-pip python3-venv
python3 --version
```

</details>

### Step 3: Create a Virtual Environment (Recommended)

This keeps SlideCraft's packages separate from your system Python.

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**macOS / Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

> You'll see `(venv)` at the start of your terminal prompt when the virtual environment is active.

---

### Step 4: Install Dependencies

```bash
pip install -r requirements.txt
```

This installs all required packages:
| Package | What it does | Size |
|---|---|---|
| `flask` | Web server | ~2 MB |
| `python-pptx` | Read/write PowerPoint files | ~5 MB |
| `Pillow` | Image processing | ~10 MB |
| `opencv-python-headless` | Inpainting & video frame processing | ~40 MB |
| `easyocr` | Text detection (OCR) | ~150 MB |
| `moviepy` | Video audio handling | ~5 MB |
| `torch` | AI backend for EasyOCR | ~200-800 MB |

> **Note:** First install may take 2-5 minutes depending on your internet speed. The `torch` package is the largest (~200MB CPU-only, ~800MB with GPU).

If you get an error with `pip`, try:
```bash
python -m pip install -r requirements.txt
```

## Usage Guide

### 1. Upload a Presentation

- Click **Upload** in the header, or **drag & drop** a `.pptx` file anywhere on the page
- Slides are converted to images and loaded into the editor

### 2. Remove NotebookLM Logo

- Click **Remove Logo** in the header
- All slides are processed — the logo is seamlessly removed using OpenCV inpainting

### 3. Edit Text with OCR

1. Click **OCR** in the toolbar — text regions are detected and highlighted with blue dashed boxes
2. Click any text region — it creates a background cover + editable text overlay
3. Edit the text in the **Properties** panel (right side)
4. Adjust font, size, color, weight to match the original
5. Click **Apply** to burn changes permanently into the slide image

### 4. Manual Text Replacement (Cover & Type)

1. Select the **Cover** tool (`C` key) in the toolbar
2. Draw a rectangle over the text you want to replace
3. The tool auto-samples the background color and creates a cover
4. Type your replacement text in the Properties panel
5. Click **Apply** to make it permanent

### 5. Add Overlays

- **Text**: `T` key or toolbar button — draw a text box, type content
- **Rectangle**: `R` key — draw a colored shape
- **Image**: Click Image button — upload an image overlay
- Double-click any text overlay to edit inline on the canvas

### 6. Export

- **PPTX**: Click **PPTX** — downloads PowerPoint with all overlays as editable layers
- **PDF**: Click **PDF** — downloads multi-page PDF of all slides
- **Apply**: Click **Apply** — burns overlays into the slide images permanently

### 7. Remove Logo from Video

1. Click **Video** in the header (or go to `/video`)
2. Upload your video file (any format)
3. A preview frame loads — drag the red box over the logo
4. Click **Remove Logo** — watch real-time progress with frame count and ETA
5. Download the clean video when done

## How Logo Removal Works

SlideCraft uses **OpenCV's Telea inpainting algorithm** to remove logos:

1. A binary mask is created marking the logo region
2. The algorithm propagates pixel values from the surrounding edges inward
3. It follows the natural color gradient and texture of the background
4. Result: seamless removal with no flat bands or color artifacts

For video, this is applied to every frame with a pre-computed mask, then audio is muxed back in.

---

## How OCR Text Editing Works

1. **EasyOCR** detects text regions with bounding boxes and confidence scores
2. When you click a region, SlideCraft samples:
   - **Background color** from a strip above the text
   - **Text color** from the darkest/lightest pixels in the region
   - **Font weight** from ink density analysis (normal/semibold/bold/extrabold)
3. A cover rectangle (matching bg color) and text overlay (matching text color + weight) are created
4. You edit the text, then **Apply** burns it into the image using Pillow with the correct system font

## Troubleshooting

| Issue | Solution |
|---|---|
| OCR not working | EasyOCR downloads models on first use (~100MB). Ensure internet access. |
| Video processing slow | Expected: ~10-30 fps depending on resolution. 1080p 5min video takes ~5-8 min. |
| Fonts don't match exactly | Use the Font Family dropdown in Properties to try different fonts. |
| PPTX upload shows blank slides | Install LibreOffice for proper PPTX-to-image conversion. |
| `PYTHONIOENCODING` error | Run with `set PYTHONIOENCODING=utf-8 && python app.py` on Windows. |
